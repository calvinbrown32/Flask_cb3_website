Another cool feature of Flask is Jinja, which allows allows me to pass in information or variables using double curly brackets. This is a name passed into the template via Jinja: {{author}}

Notes on this work can be found on my Google drive, and in the code repository itself. Some important things to note: I create the Flask and Dash objects differently than most tutorials show. This is to create a Flask driven website, capable of controlling many routes, but that also allows some of those routes to contain dash apps and dashboards.
See these code snippets for a better idea:

# Create the Flask object:
server = flask.Flask(__name__),

#Create a dash object:
app = dash.Dash(__name__, server=server, url_base_pathname='/dashapp', external_stylesheets=external_stylesheets)

# Define a route using @server.route(...):
@server.route('/data_test') def data_test():
